# Ticketresolution
#### Voorbeeld van het werken met gestructureerde output
* FastAPI applicatie
* Dependency management met uv
* Structured output op basis van Pydantic modellen
* start applicatie in devcontainer `<>`
* Gebruik `uv sync` om virtuele omgeving op te zetten
* uv start de virtuele omgeving automatisch anders... `source .venv/scripts/activate`
* Start fastAPI-server met `uv run uvicorn app:app`
